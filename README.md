An interactive **Power BI Dashboard** designed to analyze credit card fraud patterns, monitor fraudulent transactions, and provide actionable business insights through interactive visualizations.

---

## 📌 Project Overview

Credit card fraud has become one of the biggest challenges in the financial industry. This project focuses on analyzing fraudulent transaction data to identify fraud trends, high-risk locations, merchant categories, and fraud types.

Using **Power BI**, the dashboard transforms raw transaction data into meaningful visualizations that help stakeholders monitor fraud activity and make informed decisions.

---

## 🎯 Objectives

- Analyze fraudulent transaction patterns.
- Identify the most common fraud types.
- Monitor monthly fraud trends.
- Compare fraudulent transaction amounts across merchant categories.
- Visualize fraud distribution across different states.
- Build an interactive dashboard for business users.

# 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Fraud Rate | Percentage of fraudulent transactions |
| Fraudulent Transactions | Total fraud cases detected |
| Fraud Transaction Amount | Total amount involved in fraudulent transactions |
| Critical Risk Transactions | High-risk fraud transactions |

# 🔍 Key Insights

- Card Not Present fraud contributes the highest fraudulent transaction amount.
- Fraud activity varies across different states.
- Monthly trends reveal periods with increased fraud incidents.
- Merchant categories such as E-commerce and Electronics are more vulnerable to fraudulent activities.
- Interactive filters allow detailed exploration of fraud patterns.
